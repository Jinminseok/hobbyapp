# 📱 MyLife - 취미 공유 및 소통 플랫폼

> 특정 취미의 업데이트 소식을 받고, 게시판과 채팅을 통해 사람들과 취미를 공유하는 안드로이드 애플리케이션입니다.

## 💡 기획 배경
평소 관심 없던 새로운 취미에 대한 정보를 얻고, 취미 생활을 더욱 다채롭게 즐길 수 있는 소통의 장을 마련하고자 개발하게 되었습니다.

## 🛠 기술 스택
* **Language:** Java
* **IDE:** Android Studio
* **Database / Backend:** Firebase (Realtime Database, Authentication)
* **OS:** Windows 11

## 📌 주요 기능

### 🧑‍💻 일반 사용자 (User)
* **회원가입 및 로그인:** Firebase Authentication을 활용한 계정 관리
* **취미 카테고리 게시판:** 영화, 도서, 음악, 실내외 스포츠 등 분야별 게시판 조회 및 작성
* **게시글 상호작용:** 게시글 찜(좋아요) 및 댓글 작성/삭제 기능
* **오픈 채팅:** 사용자들 간의 실시간 소통을 위한 오픈 채팅방 기능
* **프로필 관리:** 마이페이지에서 개인 프로필 수정 및 찜한 게시물 확인
* **고객 센터:** 관리자와의 1:1 문의 채팅 기능

### ⚙️ 관리자 (Admin)
* **공지사항 관리:** 전체 사용자를 위한 공지사항 등록
* **게시판 관리:** 특정 취미 게시물 업데이트 및 삭제
* **문의 응대:** 사용자의 1:1 문의 사항 확인 및 답변(채팅)

> **※ 상세한 앱 실행 화면 및 UI는 함께 첨부된 `MyLIfe_ppt최종.pptx` 파일에서 확인하실 수 있습니다.**

## 👨‍👩‍👦 팀원 및 역할
* **최남규:** UI/UX 디자인 및 전반적인 기능 구현
* **김성민:** Firebase DB 연동 및 오픈 채팅, 마이페이지 기능 구현
* **진민석:** 관리자 페이지 기능 및 게시판 CRUD 기능 구현

## 💡 프로젝트 회고 및 보완점
* **보완할 점:** 게시글 조회수 및 댓글 수에 따른 인기 게시글 분류 기능 추가, 사용자별 관리자 문의 채팅방 목록 분류 작업 필요.
* **배운 점:** 앱 개발과 Firebase 연동에 대한 이해도를 높일 수 있었습니다. 백엔드 직무 능력의 중요성과 팀원 간의 소통, 협업을 통해 문제를 해결해 나가는 값진 경험을 얻었습니다.
